# Adharsh Browser Auditor

A premium, enterprise-grade browser data extraction and management suite. Adharsh Browser Auditor allows you to scan, analyze, and manage browsing data across multiple platforms with a sleek, modern interface.

![Auditor Logo](https://img.shields.io/badge/Adharsh-Browser%20Auditor-blue?style=for-the-badge)
![License](https://img.shields.io/badge/license-MIT-green?style=for-the-badge)
![Python](https://img.shields.io/badge/python-3.10%2B-blue?style=for-the-badge)

## 🚀 Features

- **Multi-Browser Support**: Seamlessly extract data from Chrome, Edge, Firefox, Brave, Comet, and Atlas.
- **Full History Extraction**: Beyond the standard limits—see your entire browsing journey.
- **Interactive Downloads**: Browse your downloads and open file locations directly in File Explorer.
- **Extension Analysis**: Human-readable extension names and IDs.
- **Selective Data Management**: Clear history by time range (Last Hour, Last 24 Hours, All Time).
- **Process Safety**: Automated checks to prevent data corruption by ensuring browsers are closed during management tasks.
- **Premium UI**: Built with `customtkinter` for a modern, high-performance visual experience.

## 🛠️ Installation

### Option A: Direct Run (No Python Required)
1.  **Download the latest executable** from the [Releases](https://github.com/yourusername/adharsh-browser-auditor/releases) section.
2.  **Run `Adharsh_Browser_Auditor.exe`** directly on your Windows machine.

### Option B: Python Installation (For Developers)
1.  **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/adharsh-browser-auditor.git
   cd adharsh-browser-auditor
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   python gui.py
   ```

## 🛡️ Safety & Privacy

Adharsh Browser Auditor is designed with privacy in mind. All data extraction and management happen **locally** on your machine. No data is ever transmitted to external servers.

> [!WARNING]
> Clearing history and cache is an irreversible action. Always ensure you have backups of important data.
